Сборка и запуск:

cmake CMakeLists.txt
make
bin/main

Описание:

Вариант Б2

Базовая часть реализована полностью.

Карта описана в файле resources/rooms.
Формат:
Число комнат
Дальше идет описание каждой комнаты в следующем виде:
номер комнаты
число дверей
далее для каждой двери: номер двери(двери нумеруются с левой по часовой стрелке от 1 до 4) номер комнаты, куда ведет дверь
число врагов
далее для каждого врага: начальные координаты патрулирования(x, y) конечные координаты патрулирования(x, y)

Можно подбирать ключи, которые открывают закрытые двери. Чтобы открыть дверь, нужно подойти к ней. 
При открытии двери число ключей уменьшается на 1. 

Можно возвращаться к предыдущим комнатам, состояние комнат сохраняется (подобранные ключи не обновляются).

Когда персонаж наступает в пустоту, выводится сообщение о проигрыше. Чтобы начать заново, нужно нажать ESC и заново запустить программу.
Аналогично при достижение портала (выхода из лабиринта) выводится сообщение о победе.
Стены работают должным образом.

Дополнительная часть:

Реализован графический эффект переход между комнатами: постепенное угасание предыдущей комнаты и появление новой.
Реализованы враги, передвигающиеся по комнате по простым маршрутам. 
При соприкосновении с противником игра заканчивается и выводится сообщение о проигрыше.

В архиве присутствует видео геймплея (gameplay.mp4), демонстрирующее реализованные пункты.
